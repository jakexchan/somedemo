项目说明

本项目是关于北京理工大学珠海学院和北京师范大学珠海分校的新浪微博用户情况。
项目分为两个模块：数据采集和数据统计分析
项目地址：
数据分析：https://github.com/kabessC/DataSystem
爬虫：https://github.com/kabessC/SinaCrawler

技术相关

技术栈：Python, HTML, CSS, Javascript
使用的框架和库：
Django Web框架
 Scrapy爬虫
Bootstrap
 Jquery
可视化JS库——ECharts(百度的一个很火的开源项目，文档：http://echarts.baidu.com/)

环境

系统环境：Linux系统(Debian 8 jessie)
数据库版本：Mysql 5.7.12
依赖：
Django (1.9.4)
 Scrapy (1.0.5)
Twisted (16.1.0)
 MySQL-python (1.2.5)
numpy (1.11.0)
 pandas (0.18.0)
jieba (0.38)
